Uroborus Player

Локальный аудиопроигрыватель для Windows. Программа собрана в один исполняемый файл (.exe) и работает без предварительной установки.

Функциональность:
- Воспроизведение форматов MP3, WAV, FLAC, AAC, OGG, M4A.
- Два режима интерфейса: полноразмерный и компактный (Mini Mode).
- Глобальные горячие клавиши для управления воспроизведением из любого окна.
- Поддержка создания и редактирования плейлистов.
- Система добавления треков через Drag-and-drop (файлы и папки).
- Интеграция в контекстное меню проводника Windows.
- Автоматическая фильтрация дубликатов при добавлении в список.

Технические особенности:
Программа написана на C# (WPF). Для работы со звуком используется библиотека NAudio, для чтения тегов — TagLib. В приложении реализована логика Single Instance через именованные каналы (Named Pipes): при запуске нового файла через проводник он передается в уже открытое окно плеера. Настройки и плейлисты сохраняются в папку пользователя AppData/Roaming.
